# SkyID-example-note-dapp

Install:

`cd DAC-example-note-dapp`

`npm install`

Build:

`npx webpack`

Your app will be compiled to the `dist` folder.

`npx webpack --watch` will listen to file modifications and compile every time you change the source code.
